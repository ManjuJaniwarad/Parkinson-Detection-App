# Parkinson Detection App

An Android application that detects Parkinson’s disease using handwriting image analysis and AI.

## Features
- Splash screen
- Image upload (Camera & Gallery)
- Image preview
- Prediction result screen
- Modern UI with animations

## Tech Stack
- Kotlin
- XML UI
- TensorFlow Lite (planned)

## Future Work
- Model integration
- Explainable AI (Grad-CAM)
